# 회귀분석 개선 결과 보고서

## 1. 개선 사항 요약
- **시간대 필터링**: 주간 시간대(09:00~17:00) 생활인구만 사용하여 신호 희석 방지
- **다중공선성 해소**: PCA를 통해 거리 관련 3개 변수를 '종합_접근성지수'로 통합
- **공간 해상도 확대**: 행정동 단위와 250m 격자 단위 분석 병행

## 2. 모델 비교

| 항목 | 행정동(이전) | 행정동(개선) | 격자(신규) |
|------|------------|------------|----------|
| 표본 수 | 306 | 306 | 13004 |
| Adj-R² | 0.569 | 0.2942 | 0.4842 |
| 접근성 p-value | 0.225 | 0.0267 | 0.0000 |
| VIF 최댓값 | 12.66 | 2.48 | 4.73 |
| H1 지지 여부 | ❌ | ✅ | ✅ |

*참고: 종합_접근성지수는 값이 클수록 접근성이 나쁜(거리가 먼) 방향으로 설정됨.*

## 3. H1 검증 결과 해석
- **행정동 모델**: 접근성지수의 영향력이 유의미함.
- **격자 모델**: 접근성지수의 영향력이 유의미함.
- **결론**: 공간 단위를 세분화함에 따라 접근성의 효과가 더 뚜렷하게 나타남.
