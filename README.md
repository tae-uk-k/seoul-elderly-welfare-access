# 서울시 노인 복지시설 접근성 및 데드존 분석 (Seoul Elderly Welfare Access Analysis)

본 프로젝트는 서울시의 노인 복지시설(복지관) 접근성이 실제 65세 이상 생활인구에 미치는 영향을 분석하고, 복지 서비스의 공급이 부족한 '데드존'을 파악하기 위한 데이터 분석 프로젝트입니다.

## 📌 프로젝트 개요
- **목적**: 복지시설 접근성(거리, 교통 등)과 노인 인구 밀집도 간의 상관관계를 다각도로 분석
- **핵심 질문**: "복지시설이 가까운 지역에 실제로 노인 생활인구가 더 많이 분포하는가?"
- **주요 방법론**: 
  - 주간 시간대(09~17시) 생활인구 필터링
  - PCA(주성분 분석)를 활용한 '종합 접근성 지수' 산출
  - 행정동 및 250m 격자 단위의 OLS 회귀분석

## 📊 주요 분석 결과 (개선된 모델 기준)

### 1. 종합 접근성 지수 (PCA)
- 복지관 거리, 지하철역 거리, 버스정류소 수의 다중공선성을 해결하기 위해 PCA를 적용하여 하나의 지수로 통합하였습니다.
- **PC1 설명력**: 약 72.7%

### 2. 회귀분석 요약
- **행정동 단위**: 접근성 지수가 유의미한 영향(p=0.027)을 미치나, 인구 밀집도 변수의 영향력이 매우 강함.
- **격자 단위 (250m)**: 분석 단위를 세분화했을 때 **접근성 지수의 유의성이 극대화(p < 0.001)**되었으며, 복지관 및 교통 접근성이 좋을수록 노인 생활인구가 유의미하게 증가함을 확인하였습니다.
